## 循环和递归

1. 斐波那契数列
    ```java
    // 很容易想到递归的方法，但是这里面没有考虑到小于0的情况，not bug free
    public int fib(int n) {
        // 最好是n<=0
        if (n == 0) {
            return 0;
        }
        if (n == 1) {
            return 1;
        }
        return fib(n - 2) + fib(n - 1);
    }
    ```

    应该避免在生产环境使用递归，很容易出现溢出现象
